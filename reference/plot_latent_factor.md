# Plot spatial variation in site loadings of HMSC models

Generate and save spatial variation in site loadings of HMSC models'
latent factors as a JPEG file.

## Usage

``` r
plot_latent_factor(path_model = NULL, env_file = ".env")
```

## Arguments

- path_model:

  Character. Path to fitted `Hmsc` model object.

- env_file:

  Character. Path to the environment file containing paths to data
  sources. Defaults to `.env`.
